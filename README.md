# Forest Wildfire and Smoke Detection

## Project Overview
This project is an AI-based Forest Wildfire and Smoke Detection System developed using Deep Learning techniques. The system detects wildfire and smoke in forest areas using YOLO-based object detection models.

## Features
- Real-time wildfire detection
- Smoke detection using AI
- Deep Learning based object detection
- Image and video analysis
- Early warning support system

## Technologies Used
- Python
- OpenCV
- YOLOv8 / YOLOv9
- Flask
- NumPy
- Pandas
- Machine Learning
- Deep Learning

## Project Objective
The objective of this project is to identify wildfire and smoke at early stages to reduce environmental damage and improve forest safety.

## Applications
- Forest monitoring
- Disaster management
- Environmental protection
- Smart surveillance systems
